# Object Oriented Programming (OOP) LAB Tasks

## 📌 Project Description
This repository contains Object Oriented Programming (OOP) laboratory exercises for Week 01 and Week 02. The programs are implemented in C++ and demonstrate fundamental programming concepts such as structures, user input/output, data handling, and modular function-based design.

## 🧪 Week 01 Experiments

### Experiment 01
Write a program that declares a structure based on a Student entity. The structure includes:
- studentID (int)
- studentName (char*)
- courseCode (char*)
- courseName (char*)
- courseMarks (int)

An instance of the structure is initialized and displayed.

---

### Experiment 02
Create a Date structure containing:
- month (int)
- day (int)
- year (int)

The user enters a date in the format (12/31/2001), and the program stores and displays it in the same format.

---

### Experiment 03
Create an Employee structure containing:
- employee number (int)
- compensation (float)

The program takes input for three employees and displays their details using a separate display function.

---

## 🧪 Week 02 Experiments

### Experiment 01

Create a Date structure and perform input/output in format (12/31/2001).
